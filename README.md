# wechat-withdrawal-monitor
基于itchat库构建文件撤回监听并发送到微信助手

简单两步即可实现微信撤回查看功能，如下所示


#### 安装依赖
```shell
pip install -r requirements.txt
```

#### 运行程序
```shell
python ./main.py
```
